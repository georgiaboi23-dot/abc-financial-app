# abc-financial-game
ABC game that provides US Alphabet training with a finacial knowledge tilt.

An educational Android application designed to introduce investing vocabulary through structured alphabet learning and interactive game modes.
This application is part of the WovenMinds Studio ecosystem and focuses on foundational financial literacy concepts presented in a child-friendly format. 

---

# Overview
ABC Investing Game teaches investing-related vocabulary using:
-Alphabet-based learning
- Child-friendly definitions
- Interactive game mode
- Progress tracking
- Optional premium unlock features

This application does not provide financial advice or real-time trading functionality.

---

## Architecture
-Kotlin
- XML Layouts
- MVVM Pattern
- Single-Activity architecture
- Fragment-based Navigation
- Google Play Billing (Planned Integration)
